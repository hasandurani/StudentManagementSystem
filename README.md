# 🎓 Student Management System - Java

A simple console-based Java application to manage student records.  

## 📌 Features
- ➕ Add a Student
- 🔍 Search Student by Roll Number
- ❌ Remove Student by Roll Number
- 📋 View All Students

## 🛠 Built With
- Java
- Object-Oriented Programming (OOP)
